Unidad didáctica 2: tecnologías fundamentales de front-end

HTML

HTML (HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web. Está estandarizado por la W3C

CSS

CSS (Cascading Style Sheets), en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado
